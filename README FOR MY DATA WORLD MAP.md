# My Data World Map

This is a Week 1 data analytics project for the Inferaq Academy Data Analytics Professional Programme.

The project uses a Jupyter Notebook to explore how everyday apps collect and use data. It looks at common services such as Netflix, Spotify, Amazon, Instagram, Google Maps, Monzo, Deliveroo, WhatsApp, LinkedIn, YouTube, Google Search, and Apple Health.

## Project Aim

The aim of this project is to understand my personal data footprint and explain how apps use data to make decisions, personalise services, improve products, and support business goals.

## What the Notebook Covers

- What data analytics means  
- The four types of analytics  
- Apps I use every day  
- The types of data collected by each app  
- How companies use this data  
- Simple Python examples using lists, dictionaries, loops, and print statements  
- A short personal reflection on data, privacy, and business value

## Tools Used

- Python 3  
- Jupyter Notebook

No extra Python libraries are required.

## How to Run the Notebook

1. Open the project folder.  
2. Open the notebook in Jupyter Notebook or JupyterLab.  
3. Run the cells from top to bottom.  
4. Check that all outputs run successfully.

## Skills Demonstrated

- Writing Markdown explanations  
- Creating a Python list  
- Creating a Python dictionary  
- Using a `for` loop  
- Using `print()` statements  
- Explaining how data is collected and used  
- Connecting data analytics to real-life apps
